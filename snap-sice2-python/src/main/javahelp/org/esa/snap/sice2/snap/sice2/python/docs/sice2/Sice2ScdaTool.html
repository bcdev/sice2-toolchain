SICE2 SCDA Cloud Detection

Overview

The SICE2 SCDA Cloud Processors provide a cloud detection derived from SLSTR L1 measurements, using the SCDA algorithm described in [1].


In SNAP, two processing modes are available:

  • Cloud detection from SLSTR L1b products. This is the standard mode which should be suitable for most users.

    SCDA Processor Description (standard mode)


  • Cloud detection from SLSTR L1b single GeoTIFF mosaic files. This special mode allows the processing as a step in the GEUS 'SICE2 toolchain', described in detail in [2], which users should be familiar with. Within this processing chain, the input files are generated by various preprocessing steps.

SICE2 Snow Retrieval Processors

Overview

The SICE2 Snow Retrieval Processors provide snow and ice physical properties derived from OLCI L1 measurements, using the retrieval algorithm developed in the frame of the SICE2 project [1] and described in a corresponding ATBD [2].


In SNAP, two processing modes are available:

  • Snow retrieval from OLCI L1b products. This is the standard mode which should be suitable for most users. Optionally, a cloud mask product derived from IdePix [3] or SCDA [4] cloud classification can be applied for exclusion of cloudy pixels.

    SICE2 Snow Retrieval Processor Description (standard mode)


  • Snow retrieval from OLCI L1b single GeoTIFF mosaic files. This special mode allows the processing as a step in the GEUS 'SICE2 toolchain', described in detail in [5], which users should be familiar with. Within this processing chain, the input files are generated by various preprocessing steps.
